Practice Language

  • Python

    Python was my second step into programming, and it’s where I discovered the beauty of problem-solving. From learning basic loops to building a small calculator, it taught me how to break down complex problems into manageable steps. It also taught me how cleanliness of the code affect the output. It’s forgiving and intuitive, which makes it the perfect language for someone just starting. I’ll never forget the moment my first Python script ran without errors—it felt like magic.

  • HTML

    HTML was my starting point for web development. It felt like learning a new way to organize thoughts as I built simple web pages. From understanding tags to creating structured layouts, it was empowering to see how every piece came together to create a functioning site. It’s amazing how much the basics of HTML can achieve.

  • CSS

    CSS has been both fun and challenging. It’s the tool that turns raw HTML into something beautiful, but aligning elements and mastering layouts like flexbox and grid sometimes feels like a puzzle. Still, every time I finish a project and see how the design comes alive, it’s incredibly rewarding. CSS taught me how much creativity coding can involve.

  • JavaScript

    JavaScript opened the door to interactivity for me. Building dynamic web pages with features like dropdown menus and form validation was a real confidence booster. It was challenging to wrap my head around concepts like asynchronous functions and DOM manipulation, but every successful project made the effort worth it. JavaScript is where I started feeling like a real developer.
